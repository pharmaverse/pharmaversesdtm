# Disease Response (Lymphoma Lugano 2014)

A SDTM RS domain for lymphoma studies using Lugano 2014 response
criteria intended for examples of ADaM dataset creation

## Usage

``` r
rs_onco_lymphoma
```

## Format

A data frame with 17 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- RSSEQ:

  Sequence Number

- RSTESTCD:

  Response Assessment Short Name

- RSTEST:

  Response Assessment Name

- RSCAT:

  Category for Response Assessment

- RSMETHOD:

  undocumented field

- RSSCAT:

  undocumented field

- RSORRES:

  Response Assessment Original Result

- RSSTRESC:

  Response Assessment Result in Std Format

- RSSTAT:

  Completion Status

- RSEVAL:

  Evaluator

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- RSDTC:

  Date/Time of Response Assessment

- RSDY:

  Study Day of Response Assessment

Generated dataset.
