# Medical History

An updated SDTM MH dataset that uses data from the CDISC pilot project

## Usage

``` r
mh
```

## Format

A data frame with 28 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- MHSEQ:

  Sequence Number

- MHSPID:

  Sponsor-Defined Identifier

- MHTERM:

  Reported Term for the Medical History

- MHLLT:

  Lowest Level Term

- MHDECOD:

  Dictionary-Derived Term

- MHHLT:

  High Level Term

- MHHLGT:

  High Level Group Term

- MHCAT:

  Category for Medical History

- MHBODSYS:

  Body System or Organ Class

- MHSEV:

  Severity/Intensity

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- VISITDY:

  Planned Study Day of Visit

- MHDTC:

  Date/Time of History Collection

- MHSTDTC:

  Start Date/Time of Medical History Event

- MHDY:

  Study Day of History Collection

- MHENDTC:

  End Date/Time of Medical History Event

- MHPRESP:

  Medical History Event Pre-Specified

- MHOCCUR:

  Medical History Occurrence

- MHSTRTPT:

  Start Relative to Reference Time Point

- MHENRTPT:

  End Relative to Reference Time Point

- MHSTTPT:

  Start Reference Time Point

- MHENTPT:

  End Reference Time Point

- MHENRF:

  End Relative to Reference Period

- MHSTAT:

  Completion Status
