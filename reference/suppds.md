# Supplemental Disposition

A SDTM SUPPDS dataset from the CDISC pilot project

## Usage

``` r
suppds
```

## Format

A data frame with 9 columns:

- STUDYID:

  Study Identifier

- RDOMAIN:

  Related Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- IDVAR:

  Identifying Variable

- IDVARVAL:

  Identifying Variable Value

- QNAM:

  Qualifier Variable Name

- QLABEL:

  Qualifier Variable Label

- QVAL:

  Data Value

- QORIG:

  Origin
