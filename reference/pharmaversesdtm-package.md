# pharmaversesdtm: SDTM Test Data for the 'Pharmaverse' Family of Packages

A set of Study Data Tabulation Model (SDTM) datasets from the Clinical
Data Interchange Standards Consortium (CDISC) pilot project used for
testing and developing Analysis Data Model (ADaM) datasets inside the
pharmaverse family of packages. SDTM dataset specifications are
described in the CDISC SDTM implementation guide, accessible by creating
a free account on <https://www.cdisc.org/>.
