# SDG

An example SDG dataset

## Usage

``` r
sdg_db
```

## Format

A data frame with 5 columns:

- termchar:

  Reported Term

- sdg_name:

  Name

- sdg_id:

  Name ID

- termvar:

  Variable

- version:

  Version
