# Vital Signs for Vaccine

An example SDTM VS dataset for vaccine studies

## Usage

``` r
vs_vaccine
```

## Format

A data frame with 23 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- VSSEQ:

  Sequence Number

- VSLNKID:

  Link ID

- VSLNKGRP:

  Link Group ID

- VSTESTCD:

  Vital Signs Test Short Name

- VSTEST:

  Vital Signs Test Name

- VSCAT:

  Category for Vital Signs

- VSSCAT:

  Subcategory for Vital Signs

- VSORRES:

  Result or Finding in Original Units

- VSORRESU:

  Original Units

- VSSTRESC:

  Character Result/Finding in Std Format

- VSSTRESN:

  Numeric Result/Finding in Standard Units

- VSSTRESU:

  Standard Units

- VSEVAL:

  Evaluator

- VSLOC:

  Location of Vital Signs Measurement

- EPOCH:

  Epoch

- VSDTC:

  Date/Time of Measurements

- VSDY:

  Study Day of Vital Signs

- VSTPT:

  Planned Time Point Name

- VSTPTNUM:

  Planned Time Point Number

- VSTPTREF:

  Time Point Reference
