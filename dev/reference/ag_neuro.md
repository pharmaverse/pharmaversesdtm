# Procedure Agents Dataset-neuro

A SDTM AG dataset recording details of agents such as a PET tracer used
in procedures

## Usage

``` r
ag_neuro
```

## Format

A data frame with 13 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- AGSEQ:

  Sequence Number

- AGTRT:

  Reported Agent Name

- AGCAT:

  Category for Agent

- AGDOSE:

  Dose per Administration

- AGDOSEU:

  Dose Units

- AGROUTE:

  Route of Administration

- AGLNKID:

  Link ID

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- AGSTDTC:

  Start Date/Time of Agent
