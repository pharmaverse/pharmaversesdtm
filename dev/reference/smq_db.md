# Standardized MedDRA Queries

An example SMQ dataset

## Usage

``` r
smq_db
```

## Format

A data frame with 6 columns:

- termchar:

  Reported Term

- scope:

  Scope

- smq_name:

  Name

- smq_id:

  Name ID

- version:

  Version

- termvar:

  Variable
