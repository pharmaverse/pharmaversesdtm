# Explore SDTM Datasets

## Datasets Preview

Here is a preview of all datasets included in the
[pharmaversesdtm](https://pharmaverse.github.io/pharmaversesdtm/)
package. Organized by therapeutic area, each dataset is displayed in a
collapsible panel with filtering options for easier exploration. For
simplicity, only the first 50 rows of each dataset are shown.
